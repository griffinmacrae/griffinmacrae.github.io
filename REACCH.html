    • REACCH

    REACCH is a bioinspired robot with the purpose of cleaning up orbital debris. I worked as an electronics and controls engineer on this project. I created an electronics diagram pictured below. I also helped with programming the control sequence for testing the arm. There has been an updated design using gecko adhesion pads being tested on cylindrical objects. However, this design is proprietary and cannot be shown here.

    The image above represents the KiCad electronic schematic I created to show the electronics for this project.

    I also worked on the controls sequence and programming force sensors for learning more information on what parts of the linked arms are being subject to the most force to come up with best strategies when grabbing the debris. I'm also currently integrating a LiDAR to the main body so it can identify the distance of objects and determine when it should start trying to grab onto these objects.

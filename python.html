Python

I worked as a Teacher's Aide for USC's CS@SC program where I taught grade schoolers how to program in python (intermediate level). This included: booleans, for/while loops, data structures, creating functions, and using libraries. This opportunity helped me improve my teaching, leadership, and communication abilities.

Pictured above is a retirement calculator I programmed in Python. It calculates different ages the user can retire depending on different factors which are switched using random number generators.

I programmed four type-J thermocouples for my senior design class to test the thermal results of my group's cooling system for a motor gearbox for quadriped robotic applications. I also programmed an IMU to calculate position data of an RC car using python.
